# Available 6-Letter Domain Names

Lists of available `.com` domain names following alternating consonant-vowel patterns.

## Method

Generated using the [`brand`](https://github.com/thorwhalen/brand) Python package.
Names are 6 letters long, following CVCVCV or VCVCVC patterns, filtered by
the `few_uniques` criterion (at most 4 unique letters, and either only 1 unique
vowel or only 1 unique consonant).

Availability was checked via a two-pass method:
1. **DNS lookup** (fast) to filter out domains that resolve
2. **WHOIS verification** on DNS-negative candidates to confirm availability

**Consonants used:** `b d f g l m n p r s t v z` (13 "pronounceable" consonants)
**Vowels used:** `a e i o u y`

**Last checked:** 2026-03-28

from brand.base import batch_check_available, fewer_consonants, vowels, few_uniques
import itertools

# Example: CVCVCV with fixed vowel
def gen_cvcvcv_fixed_vowel(consonants=fewer_consonants, vowels=vowels):
    for v in vowels:
        yield from map("".join, itertools.product(
            consonants, [v], consonants, [v], consonants, [v]))

candidates = sorted(filter(few_uniques, gen_cvcvcv_fixed_vowel()))
result = batch_check_available(candidates)
