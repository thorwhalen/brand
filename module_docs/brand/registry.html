class brand.registry.Registry(name: str)[source]

A discoverable, extensible registry of named components.

Implements collections.abc.Mapping so you can iterate, index, and check membership just like a dict.

Examples

>>> r = Registry('scorers')
>>> @r.register('length')
... def length_scorer(name):
...     return len(name)
>>> list(r)
['length']
>>> r['length']('hello')
5
>>> r['length'].cost
'cheap'

Register a function. Usable as decorator with or without arguments.

Examples

>>> r = Registry('test')
>>> @r.register
... def foo(x): return x

>>> @r.register('bar', cost='expensive')
... def bar_func(x): return x * 2
