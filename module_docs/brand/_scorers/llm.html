brand._scorers.llm

LLM-based brand evaluation scorers.

Uses Claude (via the Anthropic API) to evaluate brand names on subjective qualities that are difficult to capture with rule-based scorers: overall brand appeal, connotation quality, global safety, and domain fit.

Requires the anthropic package and a valid ANTHROPIC_API_KEY.
